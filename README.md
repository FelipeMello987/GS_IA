## 📋 O que o projeto faz

Sistema de monitoramento de missão espacial em Python. Usa o modelo Llama 3.2 via Ollama para analisar dados simulados de temperatura, energia, pressão e comunicação, e gera alertas automáticos quando a missão entra em situação crítica. O operador interage com a ARIA (Artificial Response Intelligence for Aerospace) por meio de uma interface conversacional no terminal.

---

## 🛠️ Tecnologias utilizadas

- Python 3
- [Ollama](https://ollama.com/) — execução local do modelo de linguagem
- Llama 3.2 1B — modelo de linguagem para análise e respostas
- Google Colab — ambiente de execução
- Biblioteca `ollama` (Python)

---
